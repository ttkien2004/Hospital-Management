# Cách sử dụng backend
# Project Structure:
- `server.js`: file server
- `controller`: định nghĩa các hàm cho server
- `routes`: định nghĩa các API routes.
# Lưu ý khi sử dụng:
Thêm file `.env` vào folder `backend`: 
- PORT=5000
- DB_USERNAME=<your_username>
- DB_PASSWORD=<your_password>
- DB_SERVER="127.0.0.1
- DATABASE="btl2Export"
